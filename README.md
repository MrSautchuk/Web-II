# 💻 Web II - Práticas e Exercícios

Repositório destinado aos códigos, exercícios e anotações desenvolvidos durante as aulas da disciplina de Desenvolvimento Web II. 

O foco deste repositório é o aprendizado prático e a consolidação da lógica de programação e sintaxe pura da linguagen Python, sem a utilização de frameworks, bibliotecas externas ou gerenciadores de dependências complexos.

## 🚀 Tecnologias

Atualmente, o repositório foca nos fundamentos em:
* **Python** (Puro)
* *[Outras linguagens serão adicionadas conforme o avanço das aulas]*

## 📂 Estrutura do Repositório

Os arquivos estão organizados para facilitar a consulta rápida. 

* `Aulas_python/` - Diretório principal.
* `./01) Estrutura Sequencial (IO)` - Codigos do primeiro arquivo de exercícios.
* `./02) Estrutura de Decisão` - Codigos do segundo arquivo de exercícios.
* `./03) Estrutura de Repetição` - Codigos do terceiro arquivo de exercícios.
* `./Atividades em Sala` - Codigos de exercícios escreitos durante a aula.
* `./Slides` - Slides das aulas. 
* *[Novas pastas serão criadas para cada linguagem introduzida]*

## 🏃 Como executar os scripts em Python

Como os códigos são desenvolvidos em Python puro, a execução é simples e direta. Você só precisa ter o [Python instalado](https://www.python.org/downloads/) em sua máquina.

1. Clone este repositório:
   ```bash
   git clone [https://github.com/MrSautchuk/Web-II.git](https://github.com/MrSautchuk/Web-II.git)

2. Acesse o diretório clonado:
   ```bash
   cd Web-II

3. Navegue até a pasta ou arquivo desejado e execute o script pelo terminal:
   ```bash
   python nome_do_arquivo.py

> [!NOTE]
> Dependendo da configuração do seu sistema operacional, pode ser necessário utilizar o comando python3
